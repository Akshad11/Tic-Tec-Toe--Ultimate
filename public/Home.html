How to Play XOX

  • •
    The Grid: Game is played on a 3x3 grid with two players
  • •
    Player Marks: One player uses X, the other uses O
  • •
    Winning: First to get 3 marks in a row (horizontal, vertical or diagonal) wins
  • •
    Draw: If all squares are filled with no winner, the game is a draw
